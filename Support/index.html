Frequently Asked Questions

How do I install GrafenoTTK tools?

You can download our tools from our GitHub repository. Each tool includes installation instructions in its README file. For detailed instructions, please refer to our documentation or ask for help in our Discord community.

Are GrafenoTTK tools free to use?

Yes, most of our tools are open-source and free to use. Some premium features or specialized tools may require a contribution, but this will be clearly indicated on the specific tool's page.

How can I contribute to GrafenoTTK projects?

We welcome contributions from the community! You can contribute by submitting pull requests on GitHub, reporting bugs, suggesting features, or helping other users in our Discord community. For more information on contributing, check our GitHub repository.

Where can I report bugs or issues?

You can report bugs and issues on our GitHub repository by opening an issue. Please include as much detail as possible about the problem, including steps to reproduce, expected behavior, and your system configuration.
